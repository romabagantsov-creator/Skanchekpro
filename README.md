# ЧекСкан Pro — Умный финансовый помощник

## 📱 О проекте

ЧекСкан Pro — это современное веб-приложение для учёта личных расходов. Позволяет:
- Добавлять и редактировать чеки
- Анализировать расходы по категориям
- Визуализировать траты на графиках
- Экспортировать данные в CSV

## 🚀 Установка и запуск

### Простой способ:
1. Скачайте все файлы проекта
2. Откройте `index.html` в браузере
